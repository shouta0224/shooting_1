# ===shooting_1===
避けゲーが遊べます。まだシューティング要素が追加されてません。

製作者：しょうた

# 使い方 Usage
①起動
②矢印キーで主人公(鳥みたいなやつ)を動かして弾を避ける！
　Zキーで弾が打てる！
③弾に当たるとゲームオーバー！
④スペースキーまたはエンターキーを押すとリセット！
